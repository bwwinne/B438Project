Bryce Winnecke B438 Project

github repo: https://github.com/bwwinne/B438Project

A simple, networked action-RPG game.

To start: 

Run B438Project.exe in exe folder. 
First player must press LAN Host to become Host (server) client.
Second player must press LAN Client using host's IPv4 address.

Controls:

Controller (preferred): 
Left stick to move, right stick to change character facing direction. R1/RB to attack.

Keyboard:
WASD/arrow keys to move, space to attack.
